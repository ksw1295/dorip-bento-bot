# 충남도립대 도시락 예약 자동화 (D-Bento Bot)

매일 정해진 시간에 네이버 폼으로 진행되는 도시락 예약을 자동화하기 위한 크롬 확장 프로그램입니다. 
학과, 이름 등 반복되는 입력을 자동화하고 식사 날짜를 판별하여 작동합니다.

## 주요 기능
- 설정된 식사 요일에만 자동 제출 (폼 제목의 날짜 파싱)
- 오늘 이미 예약 성공 시 중복 실행 방지
- 사용자 정보(신분, 학과, 이름, 연락처) 로컬 저장 및 자동 기입
- 예약 성공 시 달력에 식사일 기준 표시

## 설치 및 사용 방법
1. 본 레포지토리를 ZIP으로 다운로드 후 압축 해제
2. 크롬 주소창: `chrome://extensions/` 접속
3. '개발자 모드' 활성화
4. '압축해제된 확장 프로그램을 로드합니다' -> 폴더 선택
5. 확장 프로그램 팝업에서 개인 정보 및 식사 요일 저장

## 주의사항 및 면책조항
- 본 프로그램은 개인적인 편의를 위해 제작되었으며, 상시 유지보수 및 업데이트를 보장하지 않습니다.
- 폼 구조 변경이나 네트워크 환경에 따라 정상 작동하지 않을 수 있습니다.
- 모든 정보는 브라우저 내부 저장소에만 보관되며 별도의 서버로 수집되지 않습니다.
- 프로그램 사용으로 발생하는 예약 누락 등의 불이익은 사용자 본인에게 책임이 있습니다.
